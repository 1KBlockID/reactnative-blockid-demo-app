# reactnative-blockid-demo-app
Reactnative BlockID demo app

# Install
```bash
npm install
cd ios && pod install
```

# Start iOS Target (needs Xcode)
```bash
react-native run-ios
```

# Start Android Target (needs Android Studio)
```bash
react-native run-android
```

# Build APK for production
```bash
cd android && ./gradlew assembleRelease
```
